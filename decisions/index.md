# Decisions Index

This index lists decision records and supports later post-mortems and learning.

## How to use

- New decisions go in `decisions/` as `DEC-YYYY-MM-DD-<slug>.md`.
- Use the template: `docs/templates/decision.md`.
- Decisions are records: supersede, don’t rewrite.
