# Experiments Index

This index lists experiments and their outcomes so learning loops can be traced.

## How to use

- New experiments go in `experiments/` as `EXP-YYYY-MM-DD-<slug>.md`.
- Use the template: `docs/templates/experiment.md`.
- Experiments are records: supersede, don’t rewrite.
