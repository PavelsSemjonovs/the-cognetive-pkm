# Templates

Templates are stored as files so agents and humans can instantiate them mechanically.

- Resource: [docs/templates/resource.md](../templates/resource.md)

## How to use

1) Copy the relevant template into the target layer directory.
2) Rename using the naming convention in `document-standards.md`.
3) Fill required sections to meet `quality-rubric.md`.
4) Add links and update the corresponding `*/index.md`.
